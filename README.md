### Amazon Clone

A responsive Amazon homepage clone built using HTML and CSS. This project recreates the layout and design of Amazon to practice frontend web development and responsive design skills.

### Live Demo

Demo Link: [https://sakshipawar6901-bit.github.io/amazon-clone/](https://sakshipawar6901-bit.github.io/amazon-clone/)

### Features
* Responsive homepage layout
* Navigation bar similar to Amazon
* Banner section
* Product category sections
* Clean and modern UI design
* Built entirely with HTML and CSS

### Technologies Used
* HTML5
* CSS3
